About

Commands & Colors: Medieval Map Generator is a mobile application that allows fast and easy generation of maps for great board game Commands & Colors Medieval. You can choose between a few map types, enable/disable water terrain tiles, set armies strength and way how to place armies on board.

After map generation, you can select single map hex and edit it terrain type or unit. It's also possible to save/load the map or save it to the mobile gallery as a picture.

Generator settings

Here you can choose 'Map type' setting, set armies values (size) and choose a way how the Application will place armies on game board.

Map type:

  • Random: all terrain tiles are placed randomly.
  • Grouped tiles: after random placement of terrain tile, there is a 75% chance that the same type of terrain will be placed in adjected hex. If successfully placed second tile, there is a 75% chance to place third the same tile in adjected hex and so on...
  • Mirror: the map is symmetrical. If 'Use water' enabled, max river length is 6 hexes.
  • Blank: only grass on map :) If 'Use water' enabled, a river will be generated.

Units placement:

  • Random: all units are placed randomly.
  • Regions balanced: units quantity (not units values) should be similar in each region.
  • Cav. flanks, inf. center: if possible, cavalry type units are placed at flanks. Infantry type units are placed at center.

Standard size map example

Numbers at top left and bottom left describes each army. 1675, 13U, 3L mean that sum of all army units is 1675, army has 13 units and 3 leaders.

Some buttons have additional actions when hold 2 sec

  • Hold MAP button to toggle display type of units(block or letters). Letters view helps when placing terrain tiles on real map.
  • Hold UNITS button to change the army for which units will be generated. Color of button indicates which army is active. White color means both armies.

Standard size map example

Some buttons have additional actions when hold 2 sec

  • Hold SAVE button to quick save generated map. It's good to save a map before play, someone can accidentally tap a wrong button.
  • Hold PIC. button to disable/enable "Screen Always On" mode. "Screen Always On" mode helps when placing terrain tiles and units on real map.

Single map hex editing

Here you can edit single board hex. Choose terrain type, rotate it, mark it (will have a red border), place unit and/or leader on it.

Numbers in bracket after unit name are (quantity of units on map / available in box). It's possible to add more units than available in box. In that case, the unit name and 'EDIT' button on generator view will change color to red.

Change units values

Here you can set value (strength) on selected unit type. It will tell the application how strong this type of unit is.

If you have additional units block from expansions, you can change 'available in box' value.

Button DEFAULT will restore units values and 'available in box' for ALL UNITS!
